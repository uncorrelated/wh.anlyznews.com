1 数式を書く

よく知られたバーゼル級数を、

\[ \sum^{\infty}_{n=1} \frac{1}{n^2} = \frac{\pi^2}{6} \]

極限の形で捉えなおして、

\[ \lim_{m \rightarrow \infty} \sqrt{ 6 \sum^{m}_{n=1} \frac{1}{n^2} } = \pi \]

Rで近似します。

1.1 近似になる理由

\(f(m) = \sqrt{ 6 \sum^{m}_{n=1} 1 / n^2 }\)と定義すると、\(f(j)-f(j+1)>f(j+1)-f(j+2)>0\)になります。

2 コード

Rのコードはこんな感じで。

f <- function(m) sqrt( 6*sum(sapply(1:m, function(n){
    1/(n^2)
})) )
f(100)
## [1] 3.132077

3 プロット

プロットもしてみましょう。

j <- 1:25
m <- matrix(sapply(j, f), nrow=1)
colnames(m) <- j
par(mar=c(4, 4, 1, 0.1))
barplot(m, ylim=c(0, 3.5), xlab=expression("j"), ylab=expression("f(j)"), space=0, col="white")
abline(h=pi, lty=3)

なかなか\(\pi\)に近づかないですね。
